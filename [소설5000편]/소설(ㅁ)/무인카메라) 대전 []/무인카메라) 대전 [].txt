1. 한밭대로 [유성터미널 -> 동부터미널] 갑천로,주공아파트 -> 종합천사
   대전시 서구 월평동
유성과 연구단지 연결하는 갑천로 우회도로에서 주공아파트 3단지 지하차도 앞에 설치. 무단횡단자 주의. 제한속도 80km.  

 
2. 32번국도 [태안 -> 대전] 공주,박정자조각공원 -> 유성,대전
   대전시 유성구 덕명동
공주,동학사 방향의 갈림길인 "박정자조각공원" 삼거리에서 약 3km. 시,도 경계를 지나며, 언덕 내리막 왼편에 전원단지마을 있고, "유성컨트리클럽" 전에 설치. 제한속도 80km.  

 
3. 4번국도 [군산 -> 경주] 논산,계룡대 -> 두계역,방동저수지,대전
   대전시 유성구 방동
논산 방향에서 대전 방향으로 갈 때, "계룡대"를 지나서 논산시와 대전시의 경계에서 약 600m에 다리가 있다. 이 다리를 건너서 우측으로 굽은 직선도로에 설치. 제한속도 80km.  

 
4. 4번국도 [군산 -> 경주] 논산,두계역 -> 방동저수지,서대전IC,대전
   대전시 유성구 방동
논산 방향에서 대전 방향으로 갈 때, "계룡대입구"를 지나 "두계역출장소"에서 오르막을 지나 언덕 정상에 "LG주유소"가 있고, 내리막 굽은 도로의 1.2km에 설치. 전방은 계속해서 내리막이다. 안전운전 할 것. 제한속도 80km.  

 
5. 호남고속도로 [순천 -> 회덕분기점] 논산IC -> 유성IC
   대전시 유성구 원내동
회덕기점 상행 18.5km. 서대전IC 갈림길 내리막에 설치. 제한속도 100km. 화물 1.5t 및 건설기계 80km. 앞차와의 거리 100m.  

 
6. 4번국도 [경주 -> 군산] 대전,서대전IC,방동저수지,두게역입구 -> 계룡대,논산
   대전시 유성구 방동
"서대전IC"를 지나 "방동저수지"의 오르막과 내리막 굽은 도로를 지나서 우측으로 굽은 도로에 "방동마을"과 "두계역"입구의 다리를 건너서 설치. 제한속도 80km.  

 
7. 4번국도 [경주 -> 군산] 서대전IC,방동저수지 -> 두계역,계룡대,논산
   대전시 유성구 방동
대전과 서대전I.C 방향에서 논산 방향으로 갈 때, "방동저수지"의 다리를 건너 600m에 설치. 전방은 굽은 도로와 마을 진,출입로여서 사고위험이 많다. 서행할 것. 제한속도 80km.  

 
8. 17번국도 [용인 -> 여수] 신탄진,청주 -> 대전터미널
   대전시 대덕구 연축동
신탄진에서 대전시 와동 옆 경부고속도로가 있고, 내리막 우굽은도로 "S-OIL"주유소 옆에 설치. 고속도로고가 밑을 지난다. 제한속도 60km.  

 
9. 4번국도 [경주 -> 군산] 옥천,세천동 -> 판암동,대전
   대전시 동구 판암동
옥천 방향에서 대전 방향으로 갈 때, 남부순환고속도로 아래를 지나 버스종점과 "동신중고교"의 좌측으로 굽은 내리막에서 약 800m의 직선도로에 설치. "SK주유소" 앞 횡단보도를 지나서 설치. 제한속도 70km.  

 
10. 호남고속도로 [순천 -> 회덕분기점] 유성IC -> 회덕분기점
   대전시 유성구 장동
회덕기점 상행 4.5km에 설치. 제한속도 100km. 화물 1.5t 및 건설기계 80km. 앞차와의 거리 100m.

11. 한밭대로 [동부터미널 -> 유성터미널] 종함청사 -> 주공아파트
   대전시 서구 월평동
"동부터미널" 방향에서 "종합청사"를 지나 현대아파트, 상록수아파트, 주공아파트 등이 있는 지하차도를 건너 200m에 설치. 주거 밀집지역의 보행자 주의. 제한속도 80km.  

 
12. 경부고속도로 [서울 -> 부산] 청원IC -> 신탄진IC
   대전시 대덕구 덕암동
서울기점 하행 139.8km.신탄진IC(140.8km)전에 좌로 굽은도로에 설치.제한속도 100km. 화물 1.5t 및 건설기계 80km. 앞차와의 거리 100m. 버스전용차로의 구간은 "서초 IC∼신탄진 IC(137km)"의 양방향이다. 시간은 토요일에는 12 : 00∼21 : 00(9시간)이며, 일요일(국경일 포함)에는 상행선 08 : 00∼23 : 00 (15시간), 하행선 08 : 00∼21 : 00 (13시간)이다. 대상차종은 9인 이상 승용,승합차량(단,9-12인승 승용,승합차량은 6인이상 탑승시 가능)이다. 버스전용차로의 위반시 벌칙은 범칙금 6만원(승용차) - 7만원(승합차)에 벌점 30점이다.
