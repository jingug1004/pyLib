1. 305지방도 [강화 -> 인천] 검단동 -> 인천서구청
   인천시 서구 백석동
강화,김포 방향에서 인천서구청 방향으로 갈 때, "검단사거리"를 지나 "독정사거리"에서 500m를 지나 "백석고등학교" 앞에 설치. 더 정확하게 표시하자면 "SK주유소"앞에 설치. 제한속도 80km.  

 
2. 서곶로 [강화 -> 인천] 검단동 -> 인천서구청
   인천시 서구 백석동
강화,김포 방향에서 인천서구청 방향으로 갈 때, "검단사거리"를 지나 "독정사거리"에서 500m를 지나 "백석고등학교" 앞에 설치. 더 정확하게 표시하자면 "SK주유소"앞에 설치. 제한속도 80km.  

 
3. 구산로 [부평 -> 남동공단] 구월동사거리 -> 남동공단
   인천시 남동구 구월동
남부소방서 옆 구월사거리에서 800m. "구산농원" 앞에서 200m 지나서 넓은도로의 내리막길에 설치. 제한속도 80km.  

 
4. 공단로 [남동IC -> 남동공단] 남동IC -> 남동공단
   인천시 남동구 논현동
인천시청,터미널 방향에서 남동공단 방향으로 갈 때, "제2경인고속도로"를 횡단하는 고가에서 약 2,5km. 남동공단 내의 "화성보일러"와 "녹십초알로에" 앞에 설치. 제한속도 80km.  

 
5. 대공원로 [서해고속도로 -> 송내지하차도] 대공원입구 -> 송내지하차도
   인천시 남동구 장수동
수원분기점(42번국도) 교차 통과하는 고가 지나 대공원지하차도 전 300m에 설치. 제한속도 80km.  

 
6. 매립지로 [매립지 -> 행주IC] 매립지 -> 행주대교
   인천시 계양구 상야동
김포,계양구청 합류 사거리에서 800m.서울외곽순환도로 전에 설치.전방 좌로 급히 굽은도로 주의. 제한속도 70km.  

 
7. 매립지로 [매립지 -> 행주IC] 메립지 -> 행주대교
   인천시 계양구 상야동
매립지도로의 두번째 설치 지점.서울외곽순환도로 에서 1.4km 지나 설치. 제한속도 70km.  

 
8. 선학로 [문학경기장 -> 선학역] 문학경기장 -> 선학역
   인천시 연수구 선학동
인천시청 방향에서 송도신도시 방향으로 갈 때, 월드컵경기장(문학운동장) 앞 사거리에서 화훼단지를 지나면서 600m 앞에 설치. 전방 500m 내에 교차로와 지하차도, 정류장 등이 있어 각별한 주의운전이 필요한 곳이다. 절대 감속할 것. 제한속도 80km.  

 
9. 원통로 [백운오거리 -> 구십정사거리] 백운오거리 -> 구십정사거리
   인천시 부평구 십정동
현대아파트 앞의 "백운오거리"를 지나서 "구십정사거리" 방향으로 갈 때, "신촌초등교"와 "부평도서관" 앞의 오르막길 지나고, 내리막 지점에서 800m 소방서 앞 설치. 제한속도 80km.  

 
10. 경인고속도로 [서울 -> 인천] 부평IC -> 서인천IC
   인천시 부평구 청천동
신월동기점 11km.부평IC 지나 설치. 제한속도 100km. 화물차 1.5t 초과 80km. 앞차와의 거리 100m.  
 
 11. 해안로 [소래 -> 인천항] 소래 -> 송도유원지
   인천시 연수구 옥련동
"외암도삼거리"입구에서 900m. "기차여행카페", "SK주유소" 지나 육교 우로굽은 도로 지나 설치. 제한속도 80km.  

 
12. 대공원로 [송내지하차도 -> 서해고속도로] 송내지핲차도 -> 대공원입구
   인천시 남동구 장수동
충성아파트 오르막 지나 우로굽은 내리막 900m 신호등앞 설치.반대차로에서 좌회전하므로 신호 준수. 제한속도 80km.  

 
13. 기타 [송도고입구 -> 해안도로] 송도고입구 -> 해안도로
   인천시 연수구 옥련동
연수동 방향에서 "송도역입구삼거리"를 지나 해안도로를 거쳐서 인천항,경인방송국(능해로),송도신도시(해안도로) 방향으로 갈 때, "송도고입구사거리(옥골사거리)"에서 내리막 500m에 설치. 제한속도 80km.  

 
14. 해안로 [인천항 -> 소래] 송도유원지 -> 송도신도시
   인천시 연수구 옥련동
육군휴양소 앞 교차로(유원지입구)에서 직선도로 1.5km에 설치. 면허시험장,남동공단 방향. 제한속도 80km.  

 
15. 42번국도 [동해 -> 인천] 시흥 -> 남동구청
   인천시 남동구 장수동
수원 방향에서 시흥시를 지나 인천 남동공단 방향으로 진입할 때, 시경계 부근의 굽은도로를 통과하여 육교를 지난다. 대공원 방향으로 가는 "장수삼거리(치야고개삼거리)"에서 굽은 내리막 400m에 설치. 시경계지점에서 약 1,5km. 제한속도 80km.  

 
16. 39국도 [부여 -> 의정부] 인천 -> 행주대교
   인천시 계양구 상야동
인천,부천 방향에서 김포,서울 방향으로 갈 때, 하수종말처리장 앞에서 약 3km. 왼편으로 "LG주유소" 두 곳을 지나 우측으로 또하나의 "LG주유소"에서 500m 앞에 설치. 화물차량으로 인한 과속, 난폭운행이 많은 지역으로 사고다발지역이다. 안전운전할 것. 제한속도 70km.  

 
17. 305지방도 [인천 -> 강화] 인천서구청 -> 검단사거리
   인천시 서구 백석동
공촌동에서 약 2km에 인천국제공항고속국도를 통과하여 600m 지나 육교앞 삼거리에 설치.학교 앞 이므로 감속 요망. 제한속도 80km.  

 
18. 서곶로 [인천 -> 강화] 인천서구청 -> 검단사거리
   인천시 서구 백석동
공촌동에서 약 2km에 인천국제공항고속국도를 통과하여 600m 지나 육교앞 삼거리에 설치.학교 앞 이므로 감속 요망. 제한속도 80km.  

 
19. 능해로 [특정사거리 -> 송도유원지] 종합운동장 -> 송도유원지
   인천시 남구 학익동
인천항,종합경기장 방향에서 경인방송국,송도유원지,소래 방향으로 갈 때, 제2경인고속도로를 횡단하는 "능애고가"를 지나 직선도로 300m에 설치. 제한속도 80km.  

 
20. 순환도로 [주물공단 -> 인천제철] 주물공단 -> 인천제철
   인천시 서구 가좌동
북항고가 내려와 횡단보도 지나 800m에 설치. 10차로의 넓은 도로. 제한속도 80km.

21. 매립지로 [행주IC -> 매립지] 행주대교 -> 매립지
   인천시 계양구 장기동
인천광역시 계양구 장기동의 매립지 방향으로 가는 도로. 김포와 계양구청 방향으로 가는 갈림길 사거리에서 600m에 설치. 제한속도 70km.  

 
22. 39국도 [의정부 -> 부여] 행주대교 -> 인천
   인천시 계양구 상야동
올림픽대로,개화IC 방향에서 부천,인천 방향으로 갈 때, 공항뒷편의 국도가 있다. "김포매립지입구"삼거리에서 약 3km. "인천국제공항고속도로"고가 아래를 통과하여 2km의 직선도로에 설치. 이 도로는 화물차량의 과속, 난폭으로 인한 사고다발지역이다. 주의운전할 것. 제한속도 70km.  

 
23. 매립지로 [행주IC -> 매립지] 행주대교IC -> 매립지
   인천시 계양구 평동
부천방향 삼거리 우회하여 인천,김포의 경계를 지나 700m 직선도로에 설치. 제한속도 70km.
