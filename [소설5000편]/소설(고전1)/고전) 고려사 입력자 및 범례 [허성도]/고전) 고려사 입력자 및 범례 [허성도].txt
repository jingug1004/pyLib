高麗史 入力 내용 범 례

총책임자 : 허성도(서울대학교 중문학과 교수)
표점자 : 서울대학교 중문학과 대학원생 
교정자 : 서울대학교 국사학과 대학원생
최후 교정본 입력자 : 민춘희(한국사 사료 연구소 연구원)
입력본 : 高麗史 木版本
입력 체계 : 高麗史 木版本의 체계와 동일

1. 본문에 사용한 표점내용
    (1) 마침표(.) 
    (2) 쉼표(,)
    (3) 물음표(?)
    (4) 느낌표(!)
    (5) 큰 따옴표 " "
    (6) 작은 따옴표 ' '
    (7) 두점표(:)
    (8) 반 두점표(;) : 의미적 병렬을 이루고 있을 때 사용 
2. 본문에 사용한 부호 및 범례 
    (1) * : 전체적으로 문제가 있는 부분에 사용
    (2) [ ] : 원본의 注를 표시한 경우에 사용
    (3) *(  ) : 원본은 빈칸으로 되어 있으나 내용상 글자가 보충되어야 할 부분에 사용
    (4) *(??) : 원본에 글자는 있으나 字形이 불분명한 경우에 사용
    (5) *?? : 원본의 글자가 HWP의 한자 사전 및 일반 사전에 없는 경우에 사용
        단, 경우에 따라서 (4),(5)의 부호가 혼용되기도 함
    (6) { } : 바로 앞에 있는 부분의 내용을 교감하는 경우에 표시  
    (7) *& : 교감대상의 漢字 수와 교감 漢字의 수가 상이한 경우에 표시 
    (8) *(+某): 원본에는 없으나 내용상 보충해야 할 경우에 표시
    (9) ○ : 새로운 문단 표시
    (10) # : 분류기호 표시

3. 자형통일
    1) 원문의 자형을 위주로 입력함을 원칙으로 한다.
    2) 類似字 중에서 원문자형과 내용상 입력할 자형이 相異한 경우에는 목판본에서 혼용하여 상용해온 한자일 경우라도 別字로 인정하여 원문의 자형을 앞에 입력하고, 내용상 입력할 한자는 교감부호인 "{ }"으로 묶어서 바로 뒤에 표시한다.
       예) *　{經}, *　{諂}, *　{鍛}, *　{淫}, *　{須}, *　{段}, *　{束} 
           *　{斂}, *　{久}, *　{寤}, *　{刀}, *　{禳}, *　{痕}, *含{舍}
           *　{　}, *　{揀}, *　{　}, *枰{秤}, *　{諫}, *　{商}, *　{糾}
           *　{穴}, *良{艮}, *　{軌}, *祗{祇}, *侯{候}, *薄{簿}, *檎{擒}
           *　{肱}, *藉{籍}, *　{鉞}, *　{檢}, *　{刊}, *釣{鈞}, *　{塚}
           *　{暘}, *揚{楊}, *　{第}, *　{總}, *　{脅}, *陜{陝}, *　{切} 등
다만, 내용상 교감한자와 피교감자의 위치가 바뀌는 경우도 있음.
    3) 부분 이체자의 관계로써 혼용되는 경우에는 원문의 자형을 그대로 입력하고 바로 앞에 "*"로 표시한다.  
       예)  *閒 : 대부분 間의 의미로 쓰임.
            *　 : 答
            *大 : 太
            *　 : 叫 등
다만, 고유명사인 경우에는 단위로 묶거나, 교감하지 않은 경우도 있음.
    4) 일반 사전에는 없으면서 HWP사전내에 포함되어 있는 한자의 경우에는 "*원문자형" 및 "*원문자{교감자}"로 표시한다. 
       예) *　, *　{銜}, 　{馱} 등
다만, *　{齎}의 처리에 있어서 　는 齎의 이체자로서 사전에 있는 것으로 작업후에 확인되었으나 기존에 입력한 대로 그대로 두었음.
    5) 원문자형은 무시하고 내용위주로 입력한 한자의 경우
       예) 휼(   )/越 : 越로 입력.
           권(   )/券 : *券으로 입력.
           　/錫 : 각각 양/석의 字音으로 되어 있으나 내용위주인 錫으로 입력.
           　/鍊 : 각각 동/련의 字音으로 되어 있으나 사전에는 두자의 사이에 이체자 관계도 포함되어 있음. 그러므로 鍊으로 입력.
    6) HWP 한자사전의 문제점으로 인한 입력내용
       玆(1순위)　(2순위), 穀(1순위)　(2순위) -- 자형의 차이가 없음으로 각기 1순위 자형으로 통일 시켰음. 
       揷(1순위)　(2순위), 簒(1순위)　(2순위) -- 자형의 차이는 없지만 혼용되어 입력되어 있음.
       沈(1순위)　(2순위) -- 2순위 한자의 도안잘못으로 1순위 한자로 통일 시켰음.
       凉(1순위)　(2순위) -- SFT는 相異하나 PFT가 　으로써 동일함. SFT위주로 입력시켰음. 
       塚(1순위) --  SFT와  PFT가 相異함.
       亐 -- 2순위 한자사전으로 추가된 자형이나, (아마도) 1순위 한자코드로 반영되면서 기존의 1순위 한자"    (울)"이 사라진 듯함. 그러므로 사라진 한자는 *于로 代入시켰음.
    7) 기존 버전의 한자사전과 多音字의 字音 반영의 차이가 있으니 찾아보기를 할 때에 참조요망.
       예) 거란의 경우 예전에는 계란(契丹)으로만
                       현재는   글란(契丹) 글단(契丹) 계단(契丹)으로도 가능.
    8) HWPFONT의 미수용한자는 별도 목록을 작성하였으니 별첨을 참조하십시오.
       明文漢韓大字典(1991.12 明文堂)과 東亞漢韓大辭典(1996.1 東亞出版社) 참고
