[ 테스트 결과 ]
---------------------------------------------------------

*1* 무의식의 세계 깊은 곳에 있는 용궁, 그것은 당신의 마음
속에 있는 성에대한 생각입니다. 더욱이 용궁 안에는 무엇이
있는지 알 수가 없지요.
신기하고 재미있지만 위험도 가득합니다. 안으로 들어가
보지않으면 도무지알 수 없습니다. 이것은 당신이 사랑하는 사람과
처음으로 일체가 되었을때의 느낌과 같은 것입니다.

무섭게 생긴 문지기 : 죄악감 / 귀신 : 불 안감/ 재미있는
물고기 : 수치심 / 공주 : 계속되는 욕망

*2* 당신의 몸에서 가장 소중히 여기고 싶 은 부분은 장점이든
단점이든 당신자신이 가장 신경을 쓰고 있는 부분이라고 할 수
있습니다. 심리학에서는 이것을 '자기 관심 도가 높다'라고 합니다.
소중히 여기고 싶은 부분과 항목은 다음 같습니다.

## 얼굴 : 자기 자신 / 머리 : 지성 / 머 리카락 : 젊음 /
   심장 : 생명/소화기 : 건강 / 치아 : 청결함 / 눈 : 외견 /
   입 : 대화 또는 먹는 것/코: 무드 / 발 : 안정성 또 는 스피드 /
   다리 : 자신의 스타일/손 : 활동력 /허리 : 체력 또 는 비만

*3* 연인이 어느 날 당신을 두고 떠나가 버 린다면 그것은 삶의
의미를 잃는 것과 같습니다.그때 삶의 힘을 가져다 주는 것은
바로 '새로운 연인의 등장'이지요. 여행 중 다리를 삐었을 때
당신이 떠올린구원자는 실패나 실연으로 좌절 해 있을 때 '이런
때에 있어주었으면' 하는 연인의 이미지입니다.

## 자전거 : 어린 시절의 옛 친구와 같은 연인/ 오토바이 :
   정열적, 매력이 만점인 남자/ 스포츠카 : 단번 에 알 수 있는 돈
   많은 연인/ 트럭 : 순수하고 소박하며 꾸밈없는 연인/ 말 : 귀공자 타입

*4* 좋아하는 사람에게 선물하는 거울은 보 내는 사람의
시선입니다. 그것은 그 사람을 훔쳐보고 싶은 마음 인 것입니다.
늘 보고 싶은 마음이 여기에 투영되어 있습니다.

## 전신거울 : 상대방의 신체적 매력, 겉모 습이나 스타일에
   매력을 느끼는 사람/화장대 거울 : 상대방의 얼 굴에 매력을 느끼는
   사람/ 손거울 : 상대방 얼굴의 일부분에 매력 을 느끼는 > 사람/
   콤팩트 : 상대방의 내면에 매력을 느끼는 사 람

*5* 0 : 소즉적이며 욕심이 없다 / 1 : 남 성적이며 솔직하다/
    2 : 협조적이며 의존적이다 / 3 : 결속력을 중요시한다/ 7 :
    명랑하고 낙관적이다 / 99 : 항상 부족함을 느끼 고 있다/ 100 : 자기
    만족형이다

 *6* 먹고싶은 과자는 결혼 상대자로서의 최 고의 타입, 독이
들어 있다고 생각되는 과자는 결혼 상대자로서 의 최악의 타입을
말합니다.

## o : 친절하고 부드러우며 게다가 기댈 수 있는 타입/ ▽ :
   예리하고 지성적인 타입/ w : 성실, 완고, 무뚝 뚝하며 융통성이
   없는 타입/ ☆ : 멋있고 화려한 타입

*7* 연인과 데이트 시간까지 남아있는 잠깐 동안의 빈 시간.
혼자 만의 여유를 가질 수 있는 시간입니다. 그 런 때 문득 보고
싶어지는 그림은 당신의 스트레스를 나타냅 니다.

## 정물화 : 당신은 일상생활에 쫓기고 있 습니다. 늘 바쁜
   당신은 생활이 스트레스의 원인입니다.
   나체화 : 생활이 단순한가요? 나체화를 바라보고 있다고 대
   답한 당신은 일상생활에 자극 을 원하는 사람입니다.
   풍경화 : 인간 관계의 스트레스로부터 벗어나고 싶어하는 사
   람입니다.
   인간관계에 지쳐있기 때문에 풍경화를 보고싶은 것이지요.
   추상화 : 자유롭게 행동하고 싶지만 주위의 상황이 그것을
   허락하지 않기 때문에 스트레스가 쌓여 있는것입니다.
   인물화 : 현재 당신의 주위에는 신뢰할 수 있는 사람, 의지
   할 수 있는 사람이 없지 않은 가요? 의지할 수 있는
   사람을 찾고 싶은 욕구가 당신 을 인물화 앞에 멈춰서게
   한 것입니다.
   수묵화 : 도시를 벗어나 여유있는 생활 을 하고 싶은 향수를
   느끼는 것입니다. 그러나 당신 은 도시 생활을 버릴
   수도 없습니다.
   탈출하고 싶지만 그럴 수 없는 상황 이 스트레스의
   원인입니다.
