Tweak UI 98의 개요

Tweak UI 98은 윈도우 95때와 마찬가지로 윈도우 98에서도 매우 짭짤한(?)
기능을 수행한다. 또한 힘들게 레지스트리를 고쳐야만 하는 변경사항을 마
우스 조작 몇 번으로 해결할 수 있기 때문에 매우 편리하기도 하다. Tweak
UI 98의 기능을 잘 이해하고 조작하는 법만 잘 알면 윈도우98을 조금 더 편
리하게 가꿀 수 있게 될 것이다.

참고로 Tweak UI 98의 위치는 윈도우 98 원본시디의
 \TOOLS\RESKIT\POWERTOY 의 위치에 존재한다.


설치 법

압축을 풀어놓으면 'tweakui.inf' 라는 파일을 찾을 수 있다. 이 파일에 마
우스 오른쪽 버튼을 누른 다음 나타나는 메뉴에서 [설치]를 선택해 주면 된
다.

실행 법

[시작] → [설정] → [제어판]을 실행하자. 또는 빠른 방법으로 '윈도우 키
' + 'S' + 'C'를 차례대로 누르면 제어판이 슝~ 하고 나타난다. 제어판 안
에 있는 Tweak UI 아이콘을 실행시키자. 그러면 Tweak UI 98 이 그 모습을
드러낼 것이다.

[마우스(Mouse)] 탭

① Menu speed
메뉴가 열리는 속도를 조절하는 메뉴이다. 마우스로 눈금자를 왼쪽으로 드
래그하여 'fast'로 설정을 해두면 원래의 열리는 속도보다 더 빠르게 메뉴
를 열리게 할 수가 있다. 여기서 메뉴란 시작 메뉴를 비롯한 윈도우 98의
모든 응용 프로그램의 메뉴를 가리킨다.

② Mouse sensitivity
마우스의 감도를 테스트하는 메뉴이다. 여기서 마우스의 더블클릭이나 드래
그를 시험해 볼 수 있다. 오른쪽에 보이는 아이콘을 더블 클릭 하거나 마우
스로 끌어서 마우스의 감도를 테스트 해보도록 하자. (여담인데 이 아이콘
으로 마우스로 드래그 해본 사람이 과연 몇이나 될까요 ^^)

③ Use mouse wheel for scrolling
마우스의 휠 기능을 지정하는 옵션이다. 요새 시중에 판매되는 고급 마우스
들은 왼쪽 버튼과 오른쪽 버튼 사이의 가운데에 손가락으로 돌릴 수 있는
휠(wheel)을 가지고 있는 것들이 많다. 이것이 하는 기능은 긴 문서나 웹
문서의 내용을 보고자 할 때 창의 오른편에 있는 수직 스크롤 바를 움직이
게 해준다. 마우스로 스크롤 바를 일일이 움직일 필요가 없어서 요즈음 많
은 이들로부터 좋은 반응을 얻고 있는 기능이다.

Scroll a page at a time : 한번의 휠 스크롤로 한페이지씩 넘기게 해준다.
Scroll by [3] lines at a time : 한번의 휠 스크롤로 3 라인씩 넘기게 해
준다. 여기서 라인수는 사용자가 임의로 지정해 줄 수 있다.

④ Activation follows mouse(X-mouse)
이 옵션을 체크하면 X-Mouse라는 기능이 실행된다. 이것은 과거 파워 토이
에 있던 기능으로 Tweak UI98에 편입되었다. X-mouse가 하는 일은 마우스로
 가리키는 창을 무조건 활성화시키는 것이다. 그러나 생각보다 그리 편리한
 기능은 아니다.


[일반(General)] 탭

① Effects

- Windows animation
이것은 윈도우의 창이 최소화되거나 최대화 할 때의 에니메이션을 보여줄
것인지 생략할 것인지 지정해주는 옵션이다. 빠른 윈도우를 사용하고자 한
다면 이 기능을 과감히 끄자.

- Smooth scrolling
창의 스크롤을 좀더 부드럽게 하는 옵션 같은데 사용자의 체감에 그다지 영
향을 미치지 않는 것 같다.

- Beep on errors
에러가 발생했을 때 비프(Beep)음을 내게 할 것인지 정하는 옵션이다.

- Menu animation
윈도우 98에서 사용되는 응용프로그램의 메뉴 에니메이션을 사용할 것인지
지정한다. 슬라이드 식으로 열리는 메뉴들의 모습은 다소 예쁘긴 하겠지만
윈도우 98의 속도를 위해선 과감히 기능을 끄고 사용하자.


- Show Windows version on desktop
현재 사용하고 있는 윈도우의 버전과 빌드 넘버를 데스크탑의 우측 하단에
표시되도록 한다.


② Special Folders
윈도우98이 사용하는 특수 폴더들의 위치를 변경할 수 있도록 해준다. 뿐만
 아니라 이러한 기존에 사용되던 특수폴더들의 위치를 Location 박스에 표
시해 주기 때문에 파일들의 정확한 위치를 손쉽게 알 수 있다. 이러한 특수
폴더의 종류에는 Program files, 데스크탑의 파일들, 문서 저장 폴더, 즐겨
 찾기, 내 문서, 파일 보내기, 시작 메뉴 등이 있다. 그러나 이러한 특수
폴더들의 위치는 함부로 변경하지 않는 것이 좋다.

③ Internet Explorer
여기서는 검색에 사용될 기본 검색엔진을 지정해 줄 수 있다. 이렇게 지정
을 해준 다음 내 컴퓨터나 탐색기의 주소창에서 '? 검색어' 또는 '검색어
연산자(&,*,+..) 검색어' 와 같은 방식으로 웹사이트를 쉽게 검색할 수 있
다.


[탐색기(Explorer)] 탭

① Shortcut overlay
바로 가기 아이콘에 사용될 화살표의 종류를 선택할 수 있다. 기본 값은 Ar
row로 작은 화살표이다. Light arrow는 오른쪽 화살표, None은 화살표가 보
이지 않게 하며, Custom은 화살표의 종류를 사용자가 직접 지정할 수 있도
록 한다.

② Startup

- "Animated "Click here to begin"(if room)
윈도우를 시동했을 때 시작버튼을 가리키는 작은 화살표와 "Click here to
begin"이라는 메시지가 작업표시줄에 나타나도록 한다.

- Tip of the day
오늘의 팁이라는 대화상자가 표시될 것인지 아닌지를 지정한다.

③ Settings

- Prefix "Shortcut to" on new shortcuts
새로운 바로 가기 아이콘을 작성 시 "바로 가기"라는 단어를 생략할 것인지
 지정한다.

- Save Explorer windows settings
탐색기의 설정 사항은 저장해 놓는다.

- Adjust case of 8.3 filenames
파일이름을 8.3의 형식으로 보이도록 하는 옵션 같은데, 한글 윈도우에서는
 적용되지 않는 것 같다. 마찬가지로 Color of compressed files(압축된 파
일들의 색상)의 경우도 적용되지 않는 듯 하다.


[IE4] 탭

Setting이라는 커다란 옵션 아래 여러 가지 작은 옵션들이 숨어있다.

- Active Desktop enabled
액티브 데스크탑의 기능을 켜고 끄는 것이라 하는데.. 한글 윈도우98에선
힘을 못쓰는 것 같다. 켜나 끄나 사용자는 언제나 사용하던 방식으로 액티
브 데스크탑을 사용할 수 있기 때문이다.

- Add new documents to Documents on Start Menu
IE4로 열어본 웹 문서를 시작 메뉴의 문서 메뉴에 포함시키도록 하는 옵션
인 것 같다. 이때 적용되는 웹 문서는 사용자가 직접 주소를 입력한 경우라
고 하는데 ... 과연 제대로 동작할지 의문이다. 필자의 경우는 되지 않는
것 같다.

- Allow changes to Active Desktop
액티브 데스크탑 항목의 수정을 허락할 것인지 말 것인지 정한다. 만약 체
크를 해제하여 수정하지 못하도록 지정해 놓으면, 디스플레이 등록정보에서
 [웹] 탭이 사라지게 된다.

- Allow Logoff
로그오프 기능을 사용하도록 하는 기능이다. 만약 사용하지 않으면 시작메
뉴에서 '로그오프'가 사라져 버린다.

- Clear document, run, typed-URL history on exit
시스템을 종료할 때 시작 메뉴에 등록되어 있던 문서 메뉴를 깨끗이 청소하
고, 입력했던 웹 사이트 주소 기록을 모두 지우라는 옵션이다.

- Detect accidental double-clicks
사용자가 웹 페이지의 링크를 실수로 더블 클릭 해버리면 IE는 이것을 싱글
 클릭 두 번으로 인식할 수 있다. 그래서 파일을 두 번 내려 받던지 같은
문서 창을 두 개 연 다든지 하는 오동작을 유발할 수 있다. 이 옵션을 체크
해 놓으면 웹 페이지에서의 더블 클릭을 싱글 클릭으로 인식하도록 한다.
- IE4 enabled
IE4의 여러 가지 기능을 켜는 옵션이다. 만약 선택하지 않는다면 윈도우 95
때와 비슷한 기능만 수행한다는데.. 어떨지 모르겠다. ^^

- Show Documents on Start Menu
시작 메뉴에서 문서메뉴를 보이도록 한다. 만약 옵션이 체크 해제되면 시작
 메뉴에서 사라진다.

- Show Favorites on Start Menu
시작 메뉴에서 즐겨 찾기 메뉴를 보이도록 한다. 만약 옵션이 체크 해제되
면 시작 메뉴에서 사라진다.

- Show Internet icon on Desktop
데스크탑에서 인터넷 익스플로어의 아이콘을 보이도록 한다.

[DeskTop] 탭

데스크탑의 옵션창에는 바탕화면에서 보이는 여러 가지 특수 아이콘들 보이
게 할 것인지 말것인지 설정할 수 있다. 체크 표시로 간단하게 해결할 수
있기 때문에 장황한 설명은 생략한다.


[My Computer] 탭

내 컴퓨터 폴더에 보이지 않게 할 하드디스크 드라이브를 설정할 수 있다.
데스크탑 옵션과 마찬가지로 체크로 간단히 사용할 수 있기 때문에 사용하
는 데에 어려움은 없을 것이다.

[Control Panel] 탭

제어판의 구성요소들을 보여준다. 마찬가지로 제어판에 보이게 할 아이콘들
을 설정해 줄 수 있다. 그밖에 별다른 기능은 하지 않는다.

[NetWork] 탭

사용자의 시스템이 네트워크에 참여하고 있다면 윈도우 98로 부팅할 때마다
 사용자의 이름과 암호를 입력해야 할 것이다. Logon 이라는 옵션에서 'Log
 on Automatically at system startup'의 옵션을 체크해 주고 하단에 보이
는 칸에 사용자 이름과 암호를 입력해 넣자. 그러면 부팅할 때마다 자동으
로 사용자 이름과 암호가 입력될 것이다.

[New] 탭

마우스 오른쪽 버튼을 누르면 [새로 만들기]라는 명령줄이 있는 것을 발견
할 것이다. 여기에 등록된 새로 만들 항목들에 대한 설정을 가지고 있다.
이전과 마찬가지로 체크를 하고 안하고에 따라 새로 만들기의 항목으로 보
이고 안보이고를 정할 수 있다. 오른쪽 하단에 있는 [Remove] 버튼을 사용
하면 지울 수도 있다.

[Add/Remove] 탭

제어판에 있는 프로그램 추가/제거와 같은 일을 한다. 그러나 사용자는 Twe
ak UI 98에 있는 프로그램 추가/제거 기능을 절/대/로 사용하지 말 것을 당
부한다. 지워지지도 않을 뿐더라 여기서 건들면 제어판의 프로그램 추가/제
거에서도 제거할 수 없게 된다. 따라서 응용 프로그램을 삭제할 땐 반드시
제어판에 있는 '프로그램 추가/제거'를 이용하길 당부한다.

[Boot] 탭

① General

- Function Keys available
부팅할 때 F5키나 F8키를 사용할 수 있게 해준다. 윈도우95에선 'starting
windows95'라는 메시지가 나타나기 때문에 펑션키를 누를 타이밍을 알았으
나 윈도우98에서는 이런 메시지가 나타나지 않는다. 따라서 감으로 때리는
수 밖엔 없을 것이다. 그렇지 않으려면 부팅할 때 아예 CTRL 키를 누르고
있도록 한다. 그러면 부팅 메뉴가 알아서 나타날 것이다.

- Start GUI automatically
GUI모드로 부팅하게 해준다. 즉 시스템을 재 부팅하면 곧바로 윈도우가 뜨
도록 한다. 만약 그렇지 않으면 도스로 부팅하게 될 것이다.

- Display splash screen while booting
부팅할 때 윈도우 98의 로고를 보이게  할 것인지 말 것이지 정한다. 체크
해면 로고가 보이지 않게 될 것이다.

- Allow F4 to boot previous operating system
이전 도스로의 부팅을 위해서 F4키를 사용할 것이지 말 것인지를 정한다.
요새도 도스 6.x쓰는 사람이 있는가 모르겠다.

② Autorun Scandisk

윈도우 98이 비정상적으로 종료되었을 때 재부팅 시 자동으로 스캔 디스크
를 실행시킬 것인지를 지정하는 옵션이다. 기본 설정 값인 After prompting
 은 사용자에게 할 것인지 묻는 것이며 Never는 스캔 디스크 과정을 건너뛰
는 옵션이다. Without Prompting은 사용자에게 묻는 과정 없이 곧바로 스캔
 디스크를 실행시킨다.

③ Boot menu

윈도우 98이 재부팅 할 때 부팅 메뉴가 나타나게 할 것이지 말 것인지를 지
정한다. 그리고 몇초 동안 메뉴를 보이게 할 것인지를 지정 할 수 있다.

[Repair] 탭

윈도우 98의 오류난 아이콘 체계들을 다시 원래의 아이콘으로 되돌리는 기
능을 한다.

[Paranoia] 탭

① Covering your Tracks

자신의 흔적을 지우는데에 사용된다. 윈도우 98의 모든 history 흔적을 삭
제할 수 있다.

- Clear Document history at logon
로그온 시 문서 메뉴의 흔적을 청소한다.

- Clear Find Computer history at logon
로그온 시 컴퓨터 찾기의 흔적을 청소한다.

- Clear Find Files history at logon
로그온 시 파일 찾기의 흔적을 청소한다.

- Clear Internet Explorer history at logon
로그온 시 인터넷 익스플로어의 방문 흔적을 청소한다.

- Clear Last User at logon
로그온 시 마지막 사용자의 흔적을 청소한다.

- Clear Network Connection history at logon
로그온 시 네트웍 연결 흔적을 청소한다.

- Clear Run history at logon
로그온 시 실행 메뉴의 흔적을 청소한다.

- Clear Telnet history at logon
