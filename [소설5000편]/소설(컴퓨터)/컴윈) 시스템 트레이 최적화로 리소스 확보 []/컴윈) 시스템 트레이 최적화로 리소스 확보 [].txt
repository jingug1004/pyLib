

◆. 시스템 트레이 최적화
  시스템 트레이에 많은 어플리케이션이 줄줄이 띄워져 있는 분들이 많습니다.
  시스템 트레이에 표시되는 아이콘은 현재 백그라운드로 실행되고 있는
  어플리케이션을 의미하는 것입니다.
  단지, 어떤 특정 프로그램에 빠르게 접근하기 위해 시스템 트레이에 아이콘을
  만들었다면 차라리 빠른실행(Quick luncher)를 이용하는것이 바람직합니다.

  트레이에 항상 자리잡고 있는 두세 개의 아이콘이 있습니다.
  스피커와 스케줄러, 그리고 한글입력시스템 아이콘 입니다.
  도구모음 줄이 비좁고, 스케줄러도 즐겨 사용하지 않고, 사운드 카드가 아예
  없는 시스템이거나 스피커 앞에 볼륨조절 단자가 붙어 있어 직접 조절할수
  있는경우에 트레이 영역에서 아이콘을 모두 없애도 사용에 지장이 없습니다.
  오히려 윈도우가 기동하면서 쓸데 없는 프로그램을 로드하느라 걸리는 시간을
  줄일수가 있어 윈도우즈의 실행속도가 빨라지고 필요없는 장치들을 메모리로
  로드하지 않으므로 윈도우즈의 자원이 늘어납니다.
  필자의 경우는 95% 까지 늘어났습니다. 이것을 모두 없애보세요!

  1. 스케줄러 아이콘 없애기
  스케쥴러 아이콘을 더블클릭하여 연다. 고급 메뉴를 클릭하고
  '작업 스케줄러 사용 중지'를 클릭하면 된다. 다시 윈도우즈를 기동했을때
  재 실행이 된다면 [시작] - [프로그램] - [보조프로그램] - [시스템 도구]
  - [시스템 정보] 를 불러낸후 [도구] 에서 [시스템 구성 유틸리티] 를 실행
  시킨후 "시작프로그램" 항목에서 SchedulingAgent 항목의 체크를 지워주세요.

  2. 스피커 아이콘 없애기
  [제어판] -> [멀티미디어] -> [재생] 항목을 보시면 "작업 표시줄에 볼륨
  조절 표시" 항목이 있습니다. 체크를 지워주세요.

  3. 한글입력 시스템 아이콘 없애기
  [제어판] -> [키보드] -> [언어] 항목에 "작업 표시줄에 언어/한글 입력 상태
  표시기 나타내기" 의 체크를 지워주세요.

  4. 기타 다른 응용프로그램의 아이콘 없애기
  시스템 트레이에 있는 아이콘을 더블클릭 하거나 마우스 오른쪽 버튼을
  누르면 해당 유틸리티의 옵션을 줄수 있는 항목이 나옵니다.
  시스템 부팅시 적용되는 항목이나, 시스템 트레이에 적용되는 항목을 찾아서
  Disable 시키거나 stop 시키면 됩니다.

  5. 시스템 트레이에서 잘 안없어질때.
    1번 항목에서 설명한 [시스템 구성 유틸리티] 를 실행시켜
   "시작프로그램" 항목에서 해당 프로그램의 체크를 지워주시면 됩니다.

  6. 시작프로그램을 최소로 줄이자.
     [시스템 구성유틸리티] 의 '시작프로그램' 항목에 많은 부분이
    체크가 되어있다는 것은 윈도우즈를 시작할때 해당 프로그램이 모두
    로드가 된다는 소리입니다. 이 부분을 최소한만 남기고 모두 지우세요.
    꼭 필요한것은 아래와 같습니다.
    TaskMonitor
    System Tray
    입니다.
    예를들어 A4지 한장에 최대 8장까지 인쇄할수 있는 FinePrint를 사용하고
    있다면 FinePrint Dispatcher 항목은 없애면 안되겠지요.

 ☞ 위와 같이 최적화 시킨 후 윈도우즈 리소스를 검사해 보세요.
    못해도 개인별로 차이는 있겠지만 94% 정도까지 확보될것 입니다.


◆. 윈도우키를 이용한 단축버튼

  106 키보드를 사용하는 사용자는 Ctrl 키와 alt 키 사이에 놓인 윈도우
  아이콘이 그려져있는 윈도우 키를 볼 수 있습니다.
  이 키는 한번만 누르면 시작 메뉴를 빨리 열어 주는 역할도 하지만 이 키와
  다른 키를 조합하면 여러가지 단축 키의 역할도 합니다.

  아래에 윈도우 키를 조합해서 쓸수있는 단축 키를 소개합니다.

        WinKey+E                     윈도우 "탐색기" 실행
        WinKey+R                     시작 메뉴의 "실행" 메뉴
        WinKey+Pause                 시스템 등록 정보
        WinKey+F                     시작 메뉴의 "찾기" 메뉴
        WinKey+Ctrl+F                "컴퓨터" 찾기
        WinKey+M                     열려 있는 모든 창을 "최소화"
        WinKey+D                     바탕화면 보기 / 감추기
        WinKey+Shift+M               최소화한 모든 창 다시 열기
        WinKey+Tab                   작업 표시줄의 작업간 이동
        WinKey+F1                    윈도우 "도움말" 실행

◆. 마우스 오른쪽 버튼을 대신하는 "문서 키"

  106 키보드를 보면 오른쪽에 '윈도우 키'가 있습니다.
  그 윈도우키 옆에 '문서 모양에 마우스가 그려져 있는 키' 가 있습니다.
  그 키의 역할은 바로 "마우스 오른쪽 버튼"을 누른 것과 동일한 기능을 합니다.
  바탕화면이나 문서편집기, [내 컴퓨터] [시작] 버튼등에서 마우스 오른쪽
  버튼을 누른 효과와 동일한 효과를 나타냅니다.

  평소에는 마우스를 이용하므로 전혀 쓸 필요가 없는 버튼입니다만,
  만약, 마우스를 이용할 수 없는 상황에 직면하게 되었을때에는 매우 유용한
  '키'가 될것입니다.

◆. 바로가기에서 단축키를 사용

  바탕 화면에 바로 가기를 만들어 해당 응용프로그램을 즉시 실행할 수 있도록
  단축 키를 만들 수 있습니다.
  여기서 단축키를 사용하여 해당 응용프로그램이 '키보드' 조작만으로 즉시
  실행할 수 있게 만들어줍니다.

  바로 가기를 선택한 후 마우스 오른쪽 버튼을 눌러 [등록 정보]를 선택합니다.
  [바로 가는 키] 항목에서 사용자가 원하는 것은 키를 조합하여 단축키를 만들
  수 있습니다.
  조합 가능한 키는 [Ctrl + Alt ] + A 에서 Z 키 입니다.
  이렇게 단축키를 설정해 놓으면 바탕 화면에 바로가기가 보이지 않아도 실행할
  수 있습니다.

   하지만 조심해야 할 것은 현재 실행 중인 프로그램에서 쓰는 단축 키와 충돌할
   수도 있으니 단축키 배당에 주의를 기울여야 합니다.
   (예: 아래한글을 비롯한 문서편집기, 화면 캡춰 유틸리티 등등)
