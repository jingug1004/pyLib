０. 시작메뉴를 내마음대로.

   윈도우즈 98에서 보여주는 시작버튼의 설정들이 싫을때가 있습니다.
   이번에는 여길 한번 건들여 볼까요?


１. 시작 메뉴의 폴더 숨기기
  남들에게 보이고 싶지 않은 프로그램을 설치했는데 시작 프로그램에 등록되었
  다면 간단하게 안보이게 할 수 있습니다. 레지스트리 편집 같은건 처음부터
  필요도 없습니다.

  시작 버튼을 눌러 숨기길 원하는 프로그램으로 간 후 오른쪽 버튼을 눌러
  [등록 정보] 를 선택합니다. 여기서 [숨김]에 체크하면 됩니다.
  "C:\WINDOWS\시작 메뉴\프로그램"에 가서 직접 세팅을 변경할 수도 있습니다.
  숨김 속성을 해제하고 싶다면 마찬가지로 위의 폴더로 가서 숨김 속성을 해제
  시켜 주어야 합니다.

２. 시작 메뉴 최적화
  새로운 프로그램을 설치하면 그 프로그램은 시작버튼에 고유의 탭들을 만들어
  놓습니다. 때문에 이것저것 프로그램을 써보길 좋아하시는 분들이라면
  어느새 시작버튼의 길이가 화면의 길이보다 더 길어져 ▼▲ 방향키 마져
  보이게 될 순간마져 오게됩니다.
  시작 메뉴에 여기 저기 흩어져 있는 단축 아이콘을 다루기 쉽게 한번 정리해
  보지요.
  1. 시스템 유틸리티 : 각자의 이름으로 존재하는 시스템 유틸리티들은
     "시스템 유틸리티" 라는 폴더를 만들어 모두 그 하위 폴더로 저장
  2. 윈도우즈 유틸리티 : 각자의 이름으로 존재하는 윈도우즈 유틸리티들은
     "윈도우 유틸리티" 라는 폴더를 만들어 모두 그 하위 폴더로 저장
  3. 게임 관련 항목 : 각자의 이름으로 존재하는 게임프로그램들은 Games 폴더
     를 만들어 몰아 넣습니다.
  4. 마이크로 소프트 오피스를 깔면 생기는 다섯가지 정도의 프로그램도
     "마이크로소프트 오피스" 라는 폴더를 만들어 몽땅 쓸어넣습니다.
  5. 인터넷은 인터넷 끼리, 통신은 통신끼리, 압축은 압축 끼리 묶어서
     폴더를 만든후 계열끼리 그 하위 폴더에 넣어둡니다.
  6. FAQ 나 도움말, 홈페이지 접속 아이콘들은 "삭제"를 하여 보기 쉽게
     만들어 둡니다.
     삭제하는 방법은 지우길 원하는 [시작]버튼의 메뉴로 가서 마우스 오른쪽
     버튼을 눌러 "삭제"를 누르면 됩니다.

  이렇게 하면 시작 메뉴에 폴더나 단축 아이콘 깔끔하게 정리가 되어 한눈에
  보시기가 편할것 입니다.
  시작 메뉴를 여는 시간도 아낄 수 있을 뿐더러 프로그램을 실행하기 위해
  마우스를 아래 위로 스크롤하는 불편함도 줄어듭니다.
  만약 꼭 최상위 폴더에 있어야하는 아이템들이 많이있다면 시작 메뉴를 작은
  아이콘으로 표시하도록 설정해 두세요. 그러면 마우스의 이동 거리가 줄어들
  것입니다. 시작 메뉴를 작은 아이콘으로 표시하려면 작업 표시줄로 마우스를
  옮긴 후 오른쪽 버튼을 눌러 [등록 정보] -> [작업 표시줄 옵션] ->
  "시작 메뉴를 작은 아이콘으로 표시" 항목에 체크 표시를 하면 됩니다.

  ※※ 프로그램을 직접 건들이는것이 아니고 아이콘의 위치만을 변경하는 것입
       니다. 위험하지 않으니 마음껏 정리하세요.


３. [시작] 버튼 메뉴에 특별한 항목 추가하기
　
  많은 사용자들이 [제어판]에 있는 [멀티미디어]] 나 [디스플레이]를 세팅하기
  위해서는 [시작]버튼에서 3가지 단계를 거쳐가야 합니다.
  물론, 단축아이콘을 만들어 한번에 갈수도 있겠지만, 제어판에 그 많은 것들을
  모조리 바탕화면에 빼놓는다면 너무 지저분해 지겠지요?
  시작버튼에 그런것들을 보기좋고 사용하기 쉽게 등록시키는 방법을 알려드립니다.

  Windows 98 을 설치했을 때 기본적으로 나타나는 메뉴 외에 사용자가  따로
  추가할 수 있는 메뉴가 있습니다. 그러나 위에서 배운 방법대로 사용자가
  단축키를 등록시키는 방법으로는 그 특이한 기능을 온전히 쓰기에는 역부족일
  수가 있습니다. 대표적인 것들로는 [제어판], [전화접속 네트워킹], [프린터]
  와 같은것들이 있습니다. 때문에 이런 아이콘들을 시작버튼 메뉴에 추가 하려면
  특별한 CLSID가 필요합니다.

  예를들어 제어판을 시작 버튼에 추가하려면
  Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}
  라는 이름으로 폴더를 만들어 주어야 합니다.
  그냥 Control Panel만을 단축아이콘으로 만들어서 추가해서는 작동하지 않습니다.
  아래에 시작 메뉴에 추가할 수 있는 몇가지 메뉴와 관련된 CLSID가 있습니다.

  제어판
  Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}
  전화접속 네트워킹
  Dial Up Net.{992CFFA0-F557-101A-88EC-00DD010CCC48}
  프린터
  Printers.{2227A280-3AEA-1069-A2DE-08002B30309D}
  휴지통
  Recycle Bin.{645FF040-5081-101B-9F08-00AA002F954E}
  업데이트 목록
  Subscriptions.{F5175861-2688-11d0-9C5E-00AA00A45957}
  방문 사이트 목록
  History.{FF393560-C2A7-11CF-BFF4-444553540000}

  예를들어 제어판을 시작버튼의 [프로그램] 보다 위의 메뉴칸에 넣고 싶다면
  "C:\WINDOWS\시작 메뉴\프로그램"에 가서 단축아이콘을 새로 만든다음에
  '새이름' 에서 이름바꾸기를 행한후 위의 제어판 에 해당되는 문자열을 마우스로
   카피해서 넣으세요.

  이렇게 시작 메뉴에 추가한 메뉴의 이름을 사용자가 마음데로 바꾸려면 CLSID
  앞에 있는 이름을 바꾸면 됩니다.

  또한 이렇게 생성된 메뉴를 마우스로 끌어서 프로그램 메뉴와 실행메뉴 사이의
  어디에든 위치시킬 수도 있습니다.
