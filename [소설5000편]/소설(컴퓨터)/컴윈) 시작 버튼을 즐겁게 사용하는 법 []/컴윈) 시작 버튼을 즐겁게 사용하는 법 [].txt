 제  목:[TIPS] 시작버튼을 즐겁게 사용하는법                         
 올린이:x배트맨x(손무홍  ) 98/11/17 12:57  읽음:699  추천:  0 관련자료 없음

            아마 여러분은 늘 유용하게 사용하는 어플리케이션이
            적어도 네댓개 쯤은 있을 것이다. 이런 어플리케이션
            을 프로그램 메뉴에 놓아두는 대신 시작 메뉴의 위쪽
            에 쌓아둠으로써 보다 빠르게 실행할 수 있을 것이
            다. 하지만 이렇게 함으로써 시작 버튼을 누르면 길다
            랗게 늘어진 메뉴에 눈이 어지러울 것이다. 왜 마우
            스 오른쪽 버튼을 이용하지 않는가?

            시작 버튼에서 마우스 오른쪽 버튼을 누르면 새로운 
            세상이 열린다 (오우~ :-). 마우스 오른쪽 버튼을 누
            르면 탐색/열기/찾기 등 다양한 메뉴를 보게 될 것이
            다. 앞서 얘기한 레지스트리 팁을 통해 컨텐츠 메뉴
            에 여러분이 원하는 프로그램을 추가하는 방법을 배웠
            다. 이런 방법을 응용한다면 시작 버튼에 어플리케이
            션 단축 아이콘을 쌓아두는 대신 간단히 마우스 오른
            쪽 버튼을 눌러 어플리케이션을 실행할 수 있을 것이
            다. 마우스 오른쪽 버튼을 누르면 튀어 나오는 컨텐
            츠 메뉴에 어플리케이션 단축 아이콘을 추가하는 방법
            을 배워보자.

            우선 아래 키를 찾아라,

            HKEY_CLASSES_ROOT\Directory\Shell

            Shell의 바로 아래에 새로운 키를 하나 만들어 보자.
            키의 이름은 여러분이 추가하고 싶은 어플리케이션으
            로 하라. 이번엔 아래아 한글을 추가할 작정이니 HNC 
            쯤으로 이름을 지어보자. 


            여전히 키(key)와 문자열 값/바이너리 값/DWORD 값을 
            헤깔리는 사람이 있을 것인데, 그런 사람은 이 섹션
            의 앞쪽에 있는 "레지스트리란 무엇인가?"를 먼저 읽
            는게 좋을 것이다. 

            HNC 키의 오른쪽 창으로 이동하여 기본값을 더블 클릭
            하자. 값 입력창에 "&문자 제목" 형식을 채워 넣자. 
            예컨데 "&H HNC"라고 입력하면 될 것이다. & 다음에 
            오는 한개의 알파벳은 키보드를 이용하는 사람들이 빠
            르게 해당 메뉴를 선택할 수 있는 단축키 역할을 한
            다. 

            자, 다시 HNC 키로 마우스 커서를 옮긴 후 바로 아래
            에 command라는 이름을 가진 새로운 키를 하나 더 만
            들어 보자. command 키의 오른쪽 창의 기본값을 더블
            클릭하여 아래아 한글 실행 파일의 경로- c:
            \hnc\hwpw.exe를 입력하자. 

            그럼 이걸 어떻게 써먹지? 작업을 하던 중에 아래아 
            한글을 실행해야 하다면 시작 버튼을 눌러 프로그램 
            메뉴를 열고 다시 HNC 폴더를 여는 대신 시작 버튼 위
            로 마우스 커서를 옮긴 후 오른쪽 버튼을 눌러 보라.
            컨텐츠 메뉴에 HNC라는 메뉴가 나타날 것이다. 이것
            을 선택하면? 바로 아래아 한글이 실행된다.
