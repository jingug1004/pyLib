★ 파워유저로 가는 시스템 튜닝↗ 윈도우즈 98 맘 먹은데로 쓰자 ★


☞ 여러가지 item 을 다룰것이므로 순서가 없음을 양지하세요.
☞ WINDOWS 98 기준으로 작성됩니다만 WINDOWS 95에서도 될것이라 생각합니다.


０. 정말 지겨워.

  매번 윈도우즈를 종료하기 위해 [시작] -> [시스템 종료] -> [시스템 종료]
  의 3단계를 거치죠. 또, 새로운 프로그램을 인스톨 시킨 다음이나
  윈도우 레지스트리를 변경 한 후 시스템을 다시 재 작동 시켜야 할때...
  마찬가지로 3단계 작업을 하시는 분덜 많으실 겁니다.
  정말 지겨워,,,

１. 윈도우즈를 한 큐에 종료하고 싶어

  따라하세요

  1. 바탕화면에서 쥐새끼 오른쪽 눈탱이를 사정없이 눌러주시고
     [새로만들기] -> [바로가기] 를 선택해 주세요.

  2. 바로가기 작성 화면이 나오면서 명령줄에 빈 칸이 보입니다.

  3. 다음과 같이 적어주세요 (마우스로 복사해 넣으셔도 좋구요)

      rundll.exe user.exe,exitwindows

  4. 바로가기의 이름을 정합니다. 예) 한큐 종료

  5. [마침] 버튼을 누르면 끝!

  6. 바탕화면에 '한큐종료' 아이콘이 생기면 성공입니다.

    ※ 키보드에 단축키를 설정해서 끝내고 싶으시면 '한큐종료' 아이콘에서
       마우스 오른쪽 버튼을 눌러 [등록정보] 를 선택하신 후...
       '바로 가는 키' 에 마우스를 한번 찍고 원하는 키보드의 자판 하나만
       눌러주세요.
       alt 키와 ctrl 키의 조합으로 이루어 집니다.
       정했으면 '적용' '닫기' 누르면 됩니다.


２. 윈도우즈를 한 큐에 다시 시작

  따라하세요

  1. 바탕화면에서 쥐새끼 오른쪽 눈탱이를 사정없이 눌러주시고
     [새로만들기] -> [바로가기] 를 선택해 주세요.

  2. 바로가기 작성 화면이 나오면서 명령줄에 빈 칸이 보입니다.

  3. 다음과 같이 적어주세요 (마우스로 복사해 넣으셔도 좋구요)

     rundll.exe user.exe,exitwindowsexec

  4. 바로가기의 이름을 정합니다. 예) 한큐 재부팅

  5. [마침] 버튼을 누르면 끝!

  6. 바탕화면에 '한큐 재부팅' 아이콘이 생기면 성공입니다.

    ※ 키보드에 단축키를 설정해서 재부팅 하고 싶으시면 '한큐종료' 아이콘에서
       마우스 오른쪽 버튼을 눌러 [등록정보] 를 선택하신 후...
       '바로 가는 키' 에 마우스를 한번 찍고 원하는 키보드의 자판 하나만
       눌러주세요.
       alt 키와 ctrl 키의 조합으로 이루어 집니다.
       정했으면 '적용' '닫기' 누르면 됩니다.

    ※ 단지 윈도우를 재 시작해줍니다. 시스템을 종료하고 재시작 하지
       않습니다.  그러나 효과는 동일합니다.
