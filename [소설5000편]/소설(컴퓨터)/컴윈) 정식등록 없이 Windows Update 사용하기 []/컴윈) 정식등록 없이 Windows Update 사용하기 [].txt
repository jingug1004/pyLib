★ 파워유저로 가는 시스템 튜닝↗ 윈도우즈 98 맘 먹은데로 쓰자 ★


☞ 여러가지 item 을 다룰것이므로 순서가 없음을 양지하세요.


０. 그냥 해주면 안되나?

  윈도우 98은 시스템을 향상 시키기 위한 최신 드라이버와 패치 프로그램을
  마이크로 소프트 웹 사이트를 통해 WINDOWS UPDATE 라는 방식으로
  제공합니다. 이 기능은 [시작] 버튼에도 있으며 윈도우를 설치시
  바탕화면에 자동으로 설치되어 LAN 이나 전화접속 네트워킹을 통해
  다운로드 할수 있게 해줍니다.

  이것은 마치 '새 하드웨어 추가'를 통해 최신의 드라이버를 업데이트
  하는것과 비슷하죠.

  한번쯤 접속해 보신분은 아시겠지만, 이런 WINDOWS LIVE UPDATE 기능을
  이용하려면 '사용자 등록'을 해야합니다.
  정품 사용자라면 별 일 없을것이지만 한편으론 약간 귀찮은 면이 없지
  않을것 입니다. '개인정보' 유출이라는 느낌도 들것이고요.

  그냥 어디선가 '누런색 황금(?) 시디'를 구해서 쓰시는 분덜은
  정품 사용자가 아니니까 어쩔수 없을까요?

  흐흐.. 됩니다여~


１. 사용자 등록없이 Windows Update 를 받아내기


  이 작업은 레지스트리를 건들이는 작업입니다.
  자신 있으신 분은 바로 '강행' 하시고, 불안하신 분은
  '레지스트리를 편집하기 전에' 라는 강좌를 한번 쯤 읽어보신 후
  차근 차근 따라하세요.

 1. 레지스트리 편집기를 실행시킵니다.

   [시작]->[실행] 에서 'regedit.exe' 입력

 2. 아래 키를 찾습니다.

  HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion

  마치 '탐색기'에서 새로운 폴더를 만드는 것 처럼 CurrentVersion 을
  마우스를 한번 찍어주시고 '등록' 에서 '키'를 선택하신 후 'RegDone'
  이라는 문자열 값을 만듭니다. 대문자 소문자는 확실히 구분해주시고
  ' ' 는 빼고 영문자만 적어주세요 ^^:

 3. RegDone 의 기본값에서 '수정'을 선택해 '값의 데이터'를 '1'로
    바꿔줍니다.

 4. 또하나의 키를 찾아야 합니다. 만약 없다면 하나 만든다.

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Welcome\RegWiz

    대부분 정상 설치를 하셨다면 있을것 입니다만?


２. 그래서?

  이렇게 레지스트리 값을 생성 및 수정해 줌으로써 귀찮은 사용자 등록을
  하지 않고도 윈도우 98의 Windows Update 기능을 이용할 수 있습니다.
  만약, update 실행시 에러가 난다면 익스플로어 '캐쉬' 설정에서
  '캐쉬'를 완전히 비워주신 후 다시 실행해 보세요.

  네? Windows Update 아이콘을 찾을수가 없다구요?
  [시작] -> [설정] 에 가보세요. 맨 밑에 있군요!
