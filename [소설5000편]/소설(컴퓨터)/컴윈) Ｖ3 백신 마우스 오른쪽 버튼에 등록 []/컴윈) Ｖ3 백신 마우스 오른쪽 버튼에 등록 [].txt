０. 나도 V3 정식 이용자닷!

  컴퓨터를 알게되면서 부터 항상 우리 곁에 있었던 바이러스 잡는 유명한
  프로그램인 V3 라는 프로그램이 있습니다.
  개인적으로 사용시에는 사용에 제한이 없는 쉐어웨어로서 바이러스 분야에
  대해서는 최고의 프로그램이 아닌가 합니다.
  V3 에 대해서 더 말씀드리지 않겠습니다. 다 아실테니까요.

  통신상에 배포되는 V3 프로그램은 '쉐어웨어'로서 '도스용'으로 제작
  되었습니다. 물론 윈도우에서도 '도스프롬프트' 상태로 빠져나가 사용
  할 수도 있듯이 윈도우와도 아무런 충돌 없이 사용할 수 있습니다.

  그렇다고 노턴 안티바이러스나 AVP 와 같이 윈도우즈 전용 프로그램으로
  만들어 진것이 아니기때문에 윈도우 시스템과 유기적으로 연결되는 것도
  아니고 또 윈도우에서 쓰려면 약간의 손질을 해야 했습니다.

  자, 이 도스용 V3 를 윈도우 탐색기에 등록시켜 보자구요!

  네? V3 PRO 98 을 쓰신다구요?
  호호호  그럼 살포시 'P' 를 눌러주세여~


１. 마우스 오른쪽 버튼에 V3 명령 추가하기

  탐색기를 실행한 상태에서 임의의 폴더에서 마우스 오른쪽 버튼을
  눌렀을 때 'V3로 바이러스를 잡자!' 메뉴가 나타나도록 하고 싶으면
  아래 레지스트리를 먼저 찾아 수정해 주어야 합니다.

  HKEY_LOCAL_MACHINE\Software\CLASSES\Directory\shell

  ※ 쉽게 찾아지지 않을것 입니다만, 잘 찾아보세요!
     Direct x 에 관한 사항들 바로 밑에 있습니다.

  레지스트리 편집기에서 shell 에서 오른쪽 마우스를 눌러 '등록'을 선택하여
  dos prompt 라는 이름으로 새로운 키를 하나 만듭니다.
  만들어진 키의 오른쪽 창의 기본 값을 더블클릭하여 적당한 이름을 적어주세요
  예를들어 'V3 로 바이러스를 잡자!' 라고 적어주시면 됩니다.
  dos prompt 키 아래에 다시 command 라는 이름으로 새로운 키를 만듭니다.
  command 키의 오른쪽 창에 기본 값을 더블 클릭하여 값에
  'V3 . /A /S' 라고 적어줍니다. ''는 쓰지 말고 그 안에 있는 명령어만
  쓰세요 ^^:


２. 어디 한번 해볼까?

  이제 레지스트리 편집기를 종료시킨 후 윈도우 탐색기에서 임의의 폴더를
  선택한 후 마우스 오른쪽 버튼을 누르면 'V3 로 바이러스를 잡자!' 라는
  확장 메뉴가 추가되어 있을 것입니다.

  물론 이것을 실행하면 자신의 V3 프로그램으로 해당 폴더의 바이러스를
  검사해 줍니다.
  /A 는 모든 화일 검사, /S 는 서브 디렉토리까지 검사를 하라는 옵션
  입니다. 최근버전에서는 스마트하게 자동으로 서브 디렉토리까지 검사
  하는것 같습니다만, 혹시나 하는 마음에서 추가시켰습니다.

  실행시에 'V3.EXE 를 찾을수 없습니다.' 라는 메시지가 나올 수 있습니다.
  (아마 대부분 나올것 이라 생각합니다.)
  이때는 '위치지정' 탭을 눌러 자신의 V3 가 들어있는 폴더를 한번만 지정해
  주시면 다음부터는 안물어봅니다.  ^^:

  그리고 아쉽지만 A: 드라이브 나 B: 드라이브 를 지원하지는 못하고
  하드드라이브도 '전체'는 지원하지 않고 임의의 폴더와 그 하위폴더 만을
  지원합니다. 그래도 이정도면 어딥니까.... T_T

  자~ 이제 도스용 V3 프로그램도 윈도우에서 윈도우 프로그램처럼 쓰세요!
