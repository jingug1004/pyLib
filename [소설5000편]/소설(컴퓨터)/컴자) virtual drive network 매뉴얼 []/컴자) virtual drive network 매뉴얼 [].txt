           ◀◀◀ 버츄얼 드라이브 네트워크 2000 v5.0 설치법 ▶▶▶

 안녕하세요, 블루입니다. 

 요즘, 인기 최고인 '버츄얼 드라이브 2000 v5.0'을 모르시는 분은 안 계실거라 믿습
 니다. 아주 좋은 프로그램이죠. 겜시디 귀찮게 넣지 않고 하드에 이미지 떠서 바로 
 실행시킬수 있으니깐 말이죠.
 
 이 프로그램의 용도는 네크워크용으로 아주 적합하며, 물론 일반 PC사용자들에게도 
 좋습니다. 저의 매뉴얼은 영문 매뉴얼을 기초로 했으며, 많은 부분 쉽게 설명을 해
 놓았습니다. 도움이 되실꺼예요. ^^;

 참고로 공개자료실에 있는 unlimited version이라는 것은 네트워크용이 아닌 일반
 PC사용자용 입니다. 그걸 받지 마시고, 그냥 버츄얼 드라이브 2000 v5.0을 받으세요.
 evaluation version 이라 할지라도 크랙을 하시면 정품이 되니 그걸 받으시길...

 구하신 버츄얼 드라이브 네트워크 2000 v5.0을 실행시키시면 압축을 풀 폴더를 지정
 하라고 할 것입니다. 우리는 여기서 C:\Temp 폴더에 압축을 풀겠습니다.  C:\Temp 
 폴더에 보면 여러 *.doc 파일들과 한개의 vdn_inst.exe 파일, 그리고 Disk1 폴더가  
 있습니다. 먼저 vdn_inst.exe 파일을 실행시킵니다. 그러면 회사명, 시리얼 넘버, 
 압축을 풀 폴더를 묻습니다. 시리얼 넘버가 없으신 분들은 그냥 아무것도 쓰지않고  
 압축을 풀 폴더만 지정해 주시면 됩니다. 편의상 C:\vdn 폴더에 설치를 하겠습니다. 
 OK를 하고 넘어가면 압축이 풀립니다. 

 이제 C:\VDN\setup.exe 파일이 보이실 겁니다. 여기서부터 설명이 달라집니다. 

 1) 먼저 서버가 되는 컴퓨터에서의 설치법입니다. (PC방 관계자들은 주목!!!)
    
    C:\VDN\setup.exe 를 그냥 실행시키지 말고 꼭 윈도우 '시작'에서 '실행'을
    눌러준 후 setup.exe을 열고 뒤에 '/admin' 을 추가시켜 실행시키도록 합니다.
    그러고 나서, 크랙을 해줍니다. 크랙은 아주 간단하죠. 인터넷 크랙사이트에서
    쉽게 얻으실수 있습니다. 없으시면 제게 연락을.. zootblue@nownuri.net
    
    자, 크랙을 해준후 재부팅 시킵니다. 그럼 완벽한 설치가 완료되었습니다.
    이제 시스템 트레이에 생긴 버츄얼 드라이브 네트워크 2000 v5.0 아이콘에
    오른쪽 마우스 버튼을 누르면 생기는 맨 위 vdn administrator를 실행시킵니다.
    
    File탭에 가서 Cd folder 에서 New cd folder 를 하나 만들어 주고,
    File탭에 가서 Vcd files 에서 Add vcd 를 해줍니다. 공유하고자 하는 vcd파일
    을 선택해 줍니다. 
  
    닫으시면 저장할꺼냐고 묻습니다. 당근 저장하시면, 서버용 설치는 끝입니다.

 2) 둘째 클라이언트가 되는 컴퓨터에서의 설치법입니다. (PC방 관계자들은 주목!!!)
   
    꼭 서버컴의 C:\VDN\setup.exe 를 실행시키고 설치를 하셔야 합니다. 서버용과
    는 달라 뒤에 '/admin' 이라는 옵션을 달면 안됩니다. 크랙을 해주고 재부팅 
    시키면 모두 완료가 됩니다. 

    시스템 트레이에 생긴 버츄얼 드라이브 네트워크 2000 v5.0 아이콘에 오른쪽
    마우스 버튼을 누르면 서버랑은 달리 위에 두개의 항목만이 뜹니다. vdn을 클
    릭하시면 서버컴에서 설정한 vcd가 나타납니다. 실행은 쉽습니다. vcd파일을
    선택하고 insert VCD 하시면 됩니다. 
 
    이걸로 클라이언트용 설치는 끝이군요.

 3) 세째 집에서 2대의 컴을 랜으로 연결한 경우의 설치법입니다.
    
    이 부분이 중요한데, 2대의 컴을 랜으로 연결한 경우에는 서버가 되는 컴의
    C:\VDN\setup.exe 를 실행시키기 전에 C:\VDN\NoServer 폴더에 있는 setup.sys
    를 C:\VDN 으로 옮겨서 덮어씌워줘야 합니다.
    
    이후부턴 서버나 클라이언트 모두 위의 설치법과 동일합니다. 
    참, 서버컴의 C:\VDN 폴더는 지우면 안됩니다. 절대로.. 그리고, 또 한가지 
    젤 중요한거는 버츄얼 드라이브 네트워크 Administrator가 깔린 컴의 상대 컴
    에 vcd파일을 vcd로 설정해야 한다는 겁니다. (이것땜에 몇십분 헤맸음.. ^^;)

    불편하시면, 양쪽다 Administrator 버전으로 설치한다. 대신 한쪽의 vdn 폴더의
    setup.exe /admin 해서 양쪽다 설치해야 합니다.

 ※ 서버컴에서 vcd파일을 선택하고 저장해줄때마다 클라이언트 컴들의 설정도 따라
    바뀝니다. 왜 네트워크에서 이걸 써야만 하는지 확실히 아셨을 겁니다. 전 이 
    프로그램 덕에 클라이언트용 컴에 시디롬들을 살 필요가 없어져 많은 돈을 벌었
    죠. ^^; 
    
    여러분에게 유용한 정보였기를.. 그럼 이만, 바이!~
