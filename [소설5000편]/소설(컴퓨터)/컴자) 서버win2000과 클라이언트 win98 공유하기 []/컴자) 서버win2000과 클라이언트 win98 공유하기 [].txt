며칠전 어떤 분께서 메일로 질문 하신것에 대한 답변을 드렸던 내용인데 보낸 편지 함
에 보니 메일이 남아 있기에 다시 편집해 올립니다..
제가 직접 쓴 글이니 저작권 문제는 없겠죠... ^^;;;


현재 전용선을 사용하는 컴에 랜카드가 한장이 있으니 두장의 랜카드와 크로스 케이블이 필요 합니다
용산에서 리얼텍 8139 랜카드(PCI용) 사시면 1장당 약 11.000원에서 15.000정도 달라고 할겁니다
컴에 PCI 슬롯이 여유가 있다면 왠만하면 ISA 방식보다는 PCI 방식으로 구입하시는게 좋구요
크로스 케이블은 m당 한 700~800원 정도면 구입 가능하니 좀 여유있는 길이로 구입 하세요.

먼저 전용선이 연결되어 있는컴에 랜카드 한장을 더 꼽으시면 총 두장이 서버 컴에 설치가 됩니다
하나는 클라이언트 컴에 꽂으시고 그러면 서버컴 랜카드 두장, 클라이언트 컴 1장 해서
총 3장의 랜카드가 두대의 컴에 설치 됐을 겁니다..
그리고 크로스 케이블을 이용 서버컴 랜카드중 하나는 전용선 하나는 클라이언트 컴과 연결 합니다.

설정...서버 컴 (Win 2000)

1). 서버 컴의 제어판 -> 네트워크에 보시면 로컬영역과 로컬 영역2가 있습니다
    두루넷과 연결된 로컬영역에 보면 공유 항목이 있습니다. 그곳에 인터넷 연결 공유
    항목에 체크 하고 컴 이름을 지정하고 작업 그룹은 아무 이름이나 지정해 줍니다(여기서 지정하는
     이름은 클라이언트 컴에도 똑같이 적어야 합니다)
    다른것은 만지지 마시고 디폴트(Default)로 놔 두세요.
    그리고 선택한 구성요소에 IPX/SPX프로토콜과 파일/프린터 공유 프로토콜이 없다면
    설치를 누르셔서 설치를 하세요 

2). 다음 클라이언 컴과 연결된 로컬 영역의 등록 정보를 열고 TCP/IP 등록정보를 열면 일반 항목에 
    
    ① 다음 IP조소사용에 클릭 하시고 IP주소에 192.168.1.1 쓰고 서브넷 마스크에 255.255.255.0을
       쓰고 기본 게이트 웨이는 비워 둡니다
    ② 다음 DNS 서버 주소 사용에 클릭 하고 기본 설정 DNS 서버에 192.168.1.1 쓰고 
       '고급' 탭을 누른후 wins 항목에 가서 netbios 사용 안함에 체크 하세요
 
※ 이 설정도 마찬가지로 두 컴의 작업그룹(Workgroup)은 같아야 합니다..

여기까지 하셨으면 서버 컴의 설정은 모두 마친 겁니다

클라이언트 컴 (Win 98)


클라이언트 컴에는 랜카드 한장을 설치 하신후 제어판 -> 네트웍으로 가셔서 tcp/ip와 ipx/spx프로토콜과 파일/프린터 공유
프로그램 설치후tcp/ip 등록정보 누르셔서 IP주소에 할당된 IP주소 사용 선택하고
IP주소는 192.168.1.2 쓰시고 DNS구성에서 DNS사용에 선택 하고 호스트에는 1번 컴터의 이름을....
찾을 DNS서버주소에는 192.168.1.1 쓰시고 추가 버튼 누르세요

그리고 게이트웨이 에는 192.168.1.1 쓰시고 추가......
크로스 케이블로 1번 컴과 2번 컴을 연결 하세요...
 
 
여기 까지가 컴 두대를 공유하기 위한 설정은 끝 입니다..


이제 두대의 컴끼리 서로 보일수 있도록 설정 해야 합니다...

각각의 컴에 공유 하고자하는 파일이나 디렉토리를 공유 설정해 주시면 됩니다

그리고 서버 컴의 제어판 -> 사용자및 암호에 가서 클라이언트 컴을 켤때 로그온 하는
사용자 이름과 비밀 번호를 등록 하게 되면 두 컴간의 공유는 물론 인터넷 공유까지 
속도 저하 없이 완벽한 공유가 됩니다.

또 두 컴간에 스타나 기타 등등...네트워크 겜을 하려면 두 컴끼리 연결된 랜카드의 
제어판 -> 로컬영역에 IPX/SPX 등록 정보 중간 쯤에 AUTO,802.2,803.3 이런 선택 사항이 있는데
여기서 양쪽 다 802.2 를 선택해 주면 됩니다...

그러나 스타 크래프트의 경우 두 컴이 각각 배틀넷에 접속하여 겜을 즐길수는 있으나
한 방에 같이 참여는 안됩니다..
배틀넷은 네트웍 게임 중의 정보를 IP를 기준으로 주고 받기 때문에 같은 IP를 공유한
컴이 동시에 하나의 방에 진입은 안되는 것 입니다..
